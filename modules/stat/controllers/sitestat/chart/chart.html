简介

科泰 chart API 提供动态创建图表的功能。
当前具备的功能有：折线图、柱形图、饼状图
之后还会做继续的开发，相应的文档也会更新,欢迎大家保持关注
您可以在网页中使用<img>元素插入图表，当浏览器打开该网页时，chart API 提供即时图表。
本文档将介绍 chart API 的参数设置。

使用示例

返回目录 点击下面的链接会看到下面的图片，这是一张折线图
/sitestat/chart?type=lc&r=5&ma=100&mi=0&ct=01,02,03,04,05,06&ps1=40,30,56,61,43,66

下面看到的是柱状图
/sitestat/chart?type=bg&r=5&ma=100&mi=0&ct=01,02,03,04,05,06&ps1=40,30,56,61,43,66

返回目录 图片类型
